**Data Analysis with Pandas, Matplotlib & Seaborn**
This project demonstrates basic data analysis and visualization using Python.

**Tools & Libraries Used:**
Python
Pandas
NumPy
Matplotlib
Seaborn

**What I Did:**
Loaded CSV datasets using Pandas
Inspected and cleaned missing values
Applied forward-fill for missing data
Created bar charts and line plots
Visualized quantity vs unit price relationships

**Visualizations**
Horizontal bar chart using Matplotlib
Line plot using Seaborn

**Purpose**
This project was created to practice:
Data loading and cleaning,
Basic exploratory data analysis (EDA),
Data visualization with Python
